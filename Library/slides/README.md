# Slides

Index pages for slide decks — each file embeds or links to a Google Slides presentation.

You don't need to recreate your slides here. Just create a Markdown page that describes
the deck and includes the Google Slides share URL in the frontmatter `slides:` field.

**Naming:** `slides-[topic].md`
**Template:** Copy `_templates/_template-framework.md` and add the `slides:` URL.

## Ideas to add here

- Product Thinking Foundations deck
- UX Research Methods overview
- Design Systems introduction
- StayLab UX Tactics 2023 session slides
